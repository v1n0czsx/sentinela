# 🏥 Sistema Hospitalar Web

Sistema desenvolvido para simular o fluxo de atendimento de um hospital, com autenticação de usuários e cadastro de pacientes para triagem.

## ✨ Funcionalidades

- 🔐 Sistema de Login
- 👨‍⚕️ Cadastro de Pacientes
- 📄 Registro de CPF e Tipo de Atendimento
- 🔄 Integração com API REST
- 🚑 Encaminhamento para Triagem
- 🎨 Interface Responsiva e Moderna
